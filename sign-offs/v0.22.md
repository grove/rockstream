# v0.22 Sign-Off

All items from the v0.22 roadmap have been implemented and verified.

## Roadmap items

- [x] Recursive `PlanNode::Recursion { base, step, max_iterations, monotone }` and `OpKind::Recursion`
- [x] Semi-naive evaluation: each iteration feeds only the frontier (new rows) into the step function
- [x] Convergence detection: iteration halts when the frontier is empty
- [x] Safety cap: `max_iterations` prevents infinite loops on non-convergent inputs
- [x] Monotone mode (`monotone = true`): enforces insert-only; publishes `complete_through` (converged flag) once fixed-point is reached
- [x] DRed escape hatch (`monotone = false`): non-monotone deltas rejected at runtime with `RS-1509`; planner flags `not_merge_safe_reason = recursion_dred_required` in `EXPLAIN INCREMENTAL`
- [x] `WeightAdd/v1` assigned as the merge law for all Recursion nodes (abelian group; monotone declaration enables partial-progress publication)

## Error codes

- [x] `RS-1509`: Non-monotone delta rejected in monotone recursion (DRed escape hatch)
