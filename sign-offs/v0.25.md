## Deliverables

### New laws
- `BloomUnion/v1` (`MergeLawId(0x0006)`): 256-bit Bloom filter, merge = bitwise OR, semilattice, identity = all-zero 32 bytes.  Registered in `LawRegistry::with_builtins()` (6 built-in laws total).

### Plan IR additions
- `PlanNode::Lateral { input, func: LateralFunc }` — row-scoped SRF expansion.
- `LateralFunc` enum: `Unnest { col }`, `GenerateSeries { start, stop, step }`, `JsonExtractArray { col }`.
- `AggregateFunc::ApproxCountDistinct` — backed by `HyperLogLog/v1`.
- `AggregateFunc::ApproxMembership` — backed by `BloomUnion/v1`.
- `OpKind::Lateral { func }` — stateless physical operator.
- `Expr::ScalarUdf { name, args }` — scalar UDF hook (stateless).
- `UdafSpec` struct — UDAF interface sketch (not yet wired to runtime dispatch).

### Codec
- `encode_lateral_func` / `decode_lateral_func` for all three `LateralFunc` variants.
- `ScalarUdf` encode/decode added to `encode_expr` / `decode_expr`.
- `ApproxCountDistinct` / `ApproxMembership` added to `encode_agg_func` / `decode_agg_expr`.

### Explain
- `OpKind::Lateral` arm added to `rockstream-plan/src/explain.rs` and `rockstream-runtime/src/explain.rs`.

### Oracle
- `rockstream_oracle::lateral_srf_oracle` module with `LateralOracle`, `ApproxCountDistinctOracle`, `ApproxMembershipOracle`, `UdafRequirements`.
