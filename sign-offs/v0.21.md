# v0.21 Sign-Off

All items from the v0.21 roadmap have been implemented and verified.

## Roadmap items

- [x] Top-K detection with `K + epsilon` state (buffer = `2k` rows per partition)
- [x] Delete refill path (deleting a top-K row brings in the next-best from the buffer)
- [x] Delta swaps (new row outranking the k-th row emits retract + insert in one output)
- [x] Partitioned Top-K (independent top-K state per partition key)
- [x] `HyperLogLog/v1` law registered for internal cardinality estimation in planner cost model
- [x] `HyperLogLog/v1` is a semilattice: associative, commutative, idempotent (per-register max)
- [x] Escape hatch not triggered: HLL accuracy sufficient for cost-model correctness
