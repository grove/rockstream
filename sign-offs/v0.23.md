# v0.23 Sign-Off

All items from the v0.23 roadmap have been implemented and verified.

## Roadmap items

- [x] Snapshot sources: `PlanNode::Snapshot { source_name, batch_size }` and `OpKind::Snapshot`
- [x] Streamed bootstrap epochs: `SnapshotOp::next_batch()` emits at most `batch_size` insert-only rows per epoch
- [x] Bootstrap frontier: `SnapshotOp::is_complete()` signals when all rows have been delivered; `phase()` returns `BootstrapPhase::Complete`
- [x] Reconciliation after connector position loss: `SnapshotOp::resume_from(N)` skips the first N committed rows; `RS-1010` registered for position-loss interruption

## Error codes

- [x] `RS-1010`: Bootstrap interrupted; connector position lost
