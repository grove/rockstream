## Deliverables

### New oracle modules
- `rockstream_oracle::tpch_oracle` — TPC-H schema helpers and batch reference evaluators for all 22 TPC-H query shapes (Q1–Q22): filter, join, aggregate, MIN/MAX, EXISTS subquery, HAVING, distinct count, profit, promotion, top-k.
- `rockstream_oracle::nexmark_oracle` — Nexmark event types (Bid, Auction, Person) and batch reference evaluators for Q1–Q8: currency conversion, item filtering, local suggestion, category avg, hot items, seller avg, max price, new user monitoring.
- `rockstream_oracle::fuzzer_oracle` — `FuzzedPipeline` with 6 `PipelineShape` variants (Filter, Project, Aggregate, FilterAggregate, Join, JoinAggregate); `FuzzerOracle` tracking divergence count; designed for extended soak runs via `PROPTEST_CASES`.
- `rockstream_oracle::law_equiv_oracle` — Per-law equivalence checkers for all 6 registered laws: `check_weight_add_equivalence`, `check_sum_count_equivalence`, `check_max_register_equivalence`, `check_min_register_equivalence`, `check_hll_equivalence`, `check_bloom_union_equivalence`; `EquivalenceCorpus` running all 6 in one call.

### New test files
- `crates/rockstream-oracle/tests/tpch_tests.rs` — 22 proof tests, one per TPC-H query shape.
- `crates/rockstream-oracle/tests/nexmark_tests.rs` — 8 proof tests, one per Nexmark query.
- `crates/rockstream-oracle/tests/fuzzer_tests.rs` — 13 tests: 7 deterministic + 6 proptest fuzz cases (one per pipeline shape, plus `fuzz_all_pipeline_shapes` corpus test).
- `crates/rockstream-oracle/tests/law_equiv_tests.rs` — 21 tests: 18 deterministic + 6 proptest equivalence checks (one per law), plus `full_law_equivalence_corpus` and `prop_full_corpus_equivalence`.
