## Summary

Implements v0.26.0: IVM correctness freeze with law equivalence. All 22 TPC-H queries, 8 Nexmark queries, a random query fuzzer, and a law-equivalence corpus are covered by proof tests. Zero divergences found.

## Changes

### New oracle modules (`crates/rockstream-oracle/src/`)

- **`tpch_oracle.rs`**: Schema helpers (`Lineitem`, `Order`, `Customer`, `Supplier`, `Part`, `PartSupp`, `Nation`) and batch reference evaluators for all 22 TPC-H query shapes (`q1_batch` through `q22_batch`).

- **`nexmark_oracle.rs`**: Nexmark event types (`Bid`, `Auction`, `Person`) and batch reference evaluators for Q1–Q8.

- **`fuzzer_oracle.rs`**: `FuzzedPipeline` with 6 `PipelineShape` variants; `FuzzerOracle` tracking divergence count. Extended soak runs supported via `PROPTEST_CASES` environment variable.

- **`law_equiv_oracle.rs`**: Per-law equivalence checkers for all 6 registered laws; `EquivalenceCorpus` running all 6 in one call.

### New test files (`crates/rockstream-oracle/tests/`)

- **`tpch_tests.rs`**: 22 proof tests, one per TPC-H query shape (Q1–Q22).
- **`nexmark_tests.rs`**: 8 proof tests, one per Nexmark query (Q1–Q8).
- **`fuzzer_tests.rs`**: 15 tests: 8 deterministic + 7 proptest fuzz cases (one per pipeline shape).
- **`law_equiv_tests.rs`**: 21 tests: 18 deterministic + 6 proptest equivalence checks (one per law) + full corpus + large-input stress test.

### Infrastructure

- `oracle/src/lib.rs`: Added `pub mod` declarations for the 4 new modules.
- `Cargo.toml`: Version bumped 0.25.0 → 0.26.0.
- `ROADMAP.md`: v0.26 row marked `✅ Done`.
- `sign-offs/v0.26.md`: Created.

## Law-equivalence invariant (v0.26 freeze)

For all registered laws `L` and all input delta streams `{d_1, …, d_n}`:
```
L.law_merge_path(d_1, …, d_n) == L.batch_recompute(Σ d_i)
```
Verified by proptest across randomised inputs for all 6 laws (WeightAdd/v1, SumCount/v1, MaxRegister/v1, MinRegister/v1, HyperLogLog/v1, BloomUnion/v1).
