## v0.14.0: Outer, Semi, and Anti Joins

Implements the v0.14 roadmap milestone: incremental outer, semi, and anti joins using the DBSP Z-set algebraic model.

### What's new

**`OuterJoinOp`** (`rockstream-ops::outer_join`)
- `JoinType` enum: `LeftOuter`, `RightOuter`, `FullOuter`, `LeftSemi`, `LeftAnti`, `RightSemi`, `RightAnti`
- Unmatched-row state via `right_weight_by_jk` / `left_weight_by_jk` — per-join-key total weight tracking
- Zero-crossing logic: null-padded row emitted when total weight 0 → positive; retracted when positive → 0
- `process_left_delta` / `process_right_delta` / `process_epoch` with pre-change snapshot semantics
- `compact()` removes zero-weight entries; `metadata()` returns `OuterJoinMeta`

**`OuterJoinOracle`** (`rockstream-oracle::join_oracle`)
- Batch reference for seven join types using the same algebraic formula as the operator
- `NullCombineFn` and `JoinIdx` type aliases for clean API
