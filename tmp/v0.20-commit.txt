v0.20.0: Time windows, watermarks, and MaxRegister/v1

Implements tumbling time-window operator (TUMBLE) with event-time watermarks
tracked via MaxRegister/v1 (semilattice, idempotent). Escape hatch applied:
TUMBLE only; HOP/SESSION deferred to v0.21.

Key additions:
- LateDataPolicy enum (Drop, Update, RouteToSink) in rockstream-plan
- PlanNode::TumbleWindow logical plan node (time_col, window_size_ms, policy)
- OpKind::TumbleWindow physical operator (merge_law = MaxRegister/v1)
- TumbleOp in rockstream-ops: watermark-driven window closing, exactly-once
  emission, TTL-retaining emitted state, idempotent watermark advancement
- TumbleOracle batch reference in rockstream-oracle
- Catalog codec encode/decode for TumbleWindow and LateDataPolicy
- EXPLAIN support in plan, runtime, and sql crates
- DiffCtx differentiates TumbleWindow with MaxRegister/v1 merge law
- 12 proof + property tests in rockstream-oracle/tests/tumble_tests.rs:
  proof_late_data_drop_is_not_visible
  proof_late_data_update_replaces_closed_window
  proof_late_data_route_to_sink
  proof_ttl_never_removes_visible_state
  proof_tumble_closes_exactly_once_under_out_of_order_input
  prop_duplicate_watermark_replay_is_idempotent
  prop_tumble_op_matches_oracle (proptest)
  proof_tumble_window_codec_roundtrip (x3 policy variants)
  proof_diff_ctx_tumble_window_uses_max_register
  proof_watermark_is_monotone
- Version bump 0.19.0 -> 0.20.0
