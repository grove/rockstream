## Summary

Implements all v0.25 roadmap items: lateral functions/subqueries, row-scoped recomputation, scalar UDF hooks, UDAF interface sketch, and approximate-aggregate surface (`APPROX_COUNT_DISTINCT` via `HyperLogLog/v1`, `APPROX_MEMBERSHIP` via `BloomUnion/v1`).

## Changes

### New: `BloomUnion/v1` law

- `MergeLawId(0x0006)`, wire format 32 bytes (256-bit Bloom filter)
- Merge = bitwise OR, semilattice, identity = all-zero 32 bytes
- `bloom_check` / `bloom_insert` helpers for proof tests
- `LawRegistry::with_builtins()` now registers 6 built-in laws
