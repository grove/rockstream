## Summary

Implements v0.13: DBSP-sound incremental inner join with dual Z-set arrangements.

## Changes

### `rockstream-ops` — `HashJoinOp`
- Dual arrangements (`left_arr`, `right_arr`): `join_key -> (row_key, row_value) -> weight`
- Pre-change snapshot semantics: output is produced against the arrangement snapshot **before** it is updated in the same delta
- Stable row identity: `row_key` bytes are the stable identity; value bytes carry the join-key column
- `process_left_delta` / `process_right_delta`: two-phase (emit output, then update arrangement)
- `process_epoch`: combined left+right in one call
- `compact()`: GC zero-weight arrangement entries at epoch boundaries
- `metadata() -> JoinMeta`: join-key and row counts for `EXPLAIN INCREMENTAL`

### `rockstream-oracle` — `JoinOracle`
- Batch reference oracle using the bilinear join formula
- `JoinRowSchema` / `JoinRowWithValSchema`: reusable row encoding helpers for tests
