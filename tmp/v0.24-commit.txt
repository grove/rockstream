v0.24.0: View-on-view DAG

Implements the v0.24 roadmap: ViewRef, upstream view CDC, cadence
inheritance, diamond consistency, and compile-time cycle detection.

## Changes

### rockstream-types
- Add RS_1011: "View-on-view DAG contains a cycle"

### rockstream-plan
- Add PlanNode::ViewRef { view_name } — reads CDC from an upstream view
- Add OpKind::ViewRef { view_name } — physical counterpart (stateless source)
- Add dag module: detect_cycle (RS-1011), topological_order, direct_view_refs
  - detect_cycle: DFS-based compile-time cycle rejection
  - topological_order: Kahn's algorithm, deterministic scheduling order
  - direct_view_refs: collect all ViewRef names from a plan tree

### rockstream-diff
- Handle PlanNode::ViewRef -> OpKind::ViewRef with Stateless reason

### rockstream-catalog
- Codec encode/decode for PlanNode::ViewRef roundtrip

### rockstream-plan (explain), rockstream-runtime (explain)
- Explain label: ViewRef[<view_name>]

### rockstream-sql
- op_kinds and assert_op_kinds updated with ViewRef arm

### rockstream-oracle
- Add view_dag_oracle module with ViewDagOracle reference implementation
  - evaluate: topological DAG evaluation against source inputs
  - simulate_epochs: multi-epoch accumulation reference
  - has_cycle, topo_order, dependencies helpers
