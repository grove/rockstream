Prevent storage_budget hang with incomplete Azure credentials

Pre-validate that all required Azure credentials are set before calling
builder.build(), which can hang if account doesn't exist or credentials
are invalid. If AZURE_STORAGE_ACCOUNT_NAME is set but credentials are
incomplete/missing, fail fast with a clear error message instead of
hanging.

Prevents: Setting AZURE_STORAGE_ACCOUNT_NAME alone causing the benchmark
to hang indefinitely during builder initialization.
