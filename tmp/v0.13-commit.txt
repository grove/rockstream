feat(v0.13): inner joins with dual arrangements and TPC-H proof tests

Implements DBSP-sound incremental inner join (v0.13 scope):

Operator (rockstream-ops):
- HashJoinOp: dual Z-set arrangements (left_arr, right_arr)
- JoinKeyFn / CombineFn public types for join-key extraction and row
  combination
- Pre-change snapshot semantics: output computed from arrangement
  snapshot BEFORE it is updated in the same delta
- Stable row identity: row_key bytes are the stable identity; value
  bytes carry the join-key column
- process_left_delta / process_right_delta: explicit two-phase
  (emit output then update arrangement)
- process_epoch: combined left+right in one call
- compact(): GC zero-weight arrangement entries at epoch boundaries
- metadata() -> JoinMeta: join-key and row counts for EXPLAIN INCREMENTAL

Oracle (rockstream-oracle):
- JoinOracle: batch reference using the bilinear join formula
  join(L,R) = { combine(l,r), w_l*w_r | l in L, r in R, key(l)==key(r) }
- JoinRowSchema / JoinRowWithValSchema: reusable row encoding helpers
